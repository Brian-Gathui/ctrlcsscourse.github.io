CSS Can Be Quite
a Useful Skill to Learn

As an aspiring web designer deciding on which particular coding language to learn is a particularly difficult question to answer. However with website applications like SquareSpace or Wix making website development more accessible to the every day user, and tools such as SCSS compiler, Bourbon, PostCSS, and Sierra Library making CSS easier; it is becoming increasingly more rare to find website developers who have mastered stock CSS.

Without the utilization of CSS every webpage would be just plaintext and images cascading vertically down a webpage. CSS allows a coder to change colors, rearrange the order and orientation of text and images on a webpage. Giving the coder the ability to affect a webpage all the way from the macro to the micro. It allows bloggers, developers, and web designers the necessary skills that allow them to make attractive and unique websites.

Registration Quiz

This quiz is part of the Ctrl's course registration it just gives a gauge of your ability in-order to guide you appropriately. Remember to take your time even if its 5 questions

1. Which of the following is the correct format for the CSS?

  • p { color: red, text-align: center; }
  • p { color: red; text-align: center; }
  • p { color- red, text-align- center; }
  • p { color: red, text-align: center;
Good Job, You got it Right!! Keep it Up :)

Correct Answer: B

Feedback: CSS is composed of a selector and a declaration wherein the selector points to the HTML element you want to style. The declaration block contains one or more declarations separated by semicolons. Each declaration includes a CSS property name and a value, separated by a colon and declaration blocks are surrounded by curly braces.

See W3 Schools for more information.

  • _sen1 { text-align: center; color: red; }
