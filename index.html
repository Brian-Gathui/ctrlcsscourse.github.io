Confident in Your CSS Skills

CSS, (Cascading Style Sheets) along with HTML (Hypertext Markup Language) are two core languages used for building webpages. CSS, is specifically used to style an HTML document. When you open up a webpage, the layout of the page and design of the page is created using CSS.

Our objective in Ctrl is to provide courses and services to teach CSS in three levels: Beginner, Intermediate, and Advanced.

How it Works

Our CSS coding course registration process is so simple it can
be summed up into three main stages...

01: Registration Quiz

In order to customize the course to our students needs a registration quiz is necessary. Part of the course registration, the Registration quiz is 5 questions.

02: Registration Form

Fill in all the fields in the registration form, found on the syllabus page, and click submit. An email will be sent confirming registration.

02: Download and Begin

After receiving email conformation and placement, download required files and software, for example, Microsoft Visual Studio Code.
